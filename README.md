# Nutraleaves

> Nutraleaves is a modern landing page designed for a wellness and nutrition brand focused on **microgreens, antioxidants, and longevity**.  
The project emphasizes clean UI, clear content structure, and a professional design

## Project Overview
Nutraleaves aims to promote the importance of antioxidants, polyphenols, vitamins, and minerals through high-quality, organic microgreens.  
The landing page communicates trust, quality, and a science-backed approach to healthy living.

## Tech Stack
- React(vite)
- Tailwindcss
